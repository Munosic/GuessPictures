### 実行環境
terminalで二つのウィンドウを開き、
サーバー側では以下のコマンドを入力。<br>
`java -jar Oekaki.jar server`<br><br>
クライアント側では以下のコマンドを入力<br>
`java -jar Oekaki.jar client`
